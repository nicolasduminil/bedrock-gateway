Welcome to the Amazon Bedrock Sandbox

The Amazon Bedrock Sandbox is deployed !
Here are some of the features this service provides:

  • Ability to test different FMs (Foundation Models) via the Amazon Bedrock service
  • Ability to dynamically configure each LLM parameters
  • Ability to build text generators
  • Ability to build text summarizers
  • Ability to build conversational assistants and chatbots
  • The FMs suported in the EU-WEST AZ are: Titan Text G1 and Mistral 7/8
  • Claude 2/3 is also supported but require special approvals
  • Other AZs, like US-EAST-1, support additional FMs like Llama2, Jurasic2
  • Fully integrated with AWS REST Gateway
  • Fully integrated with AWS Serverless
  • Fully integrated with Quarkus
